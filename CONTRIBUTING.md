# Contributing to canihavesex.today

Thanks for your interest in contributing! We welcome contributions from everyone.

## Getting Started

1.  **Fork and Clone**: Fork the repo and clone it locally.
2.  **Install Dependencies**: Run `npm install` in the root directory.
3.  **Environment Setup**: Copy `.env.example` to `.env` and fill in required values.
4.  **Run Locally**:
    *   Backend: `npm run dev:backend`
    *   Frontend: `npm run dev:frontend`
    *   Both: `npm run dev`

See [LOCAL_DEVELOPMENT.md](./LOCAL_DEVELOPMENT.md) for detailed instructions.

## Code Quality

*   **Linting**: Run `npm run lint` in both `apps/backend` and `apps/frontend` to ensure code quality.
*   **Formatting**: Run `npm run format` to fix formatting issues.
*   **Commits**: Write clear, descriptive commit messages.

## Making Changes

1.  Create a new branch for your feature or fix.
2.  Make your changes.
3.  Verify your changes locally.
4.  Ensure there are no linting errors.
5.  Submit a Pull Request.

## Reporting Issues

Please use the provided issue templates to report bugs or request features.
