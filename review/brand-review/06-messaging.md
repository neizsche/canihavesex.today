# Brand Review: Messaging

## Title: Direct and Honest Technical Copy

**Severity:** Informational
**Confidence:** High
**Category:** Messaging
**Evidence:** 
"Behind the scenes, one open-source engine turns those signals into a single daily answer... One rule guides every call: when the signal is unclear, it assumes you're fertile."

**Current Experience:**
The messaging treats the user as an intelligent adult. It explains *how* the algorithm works in plain English rather than hiding behind "AI" buzzwords.

**Why It Matters:**
Trust is built on transparency. By explicitly stating the engine's core bias (assuming fertility when unsure), the messaging establishes immense credibility with health-conscious users.

**Suggested Direction:**
Positive finding. Maintain this level of technical transparency in all future marketing.

**Estimated Effort:** N/A
**Launch Blocker:** No
